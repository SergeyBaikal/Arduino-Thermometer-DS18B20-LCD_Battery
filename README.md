# Arduino-Thermometer-DS18B20-LCD_Battery
Arduino Nano термометр с DS18B20, LCD 16×2, контролем стабилизации, проверкой CRC и мониторингом батареи 18650



![Термометр в работе](screenshot_display.jpg)

## Возможности

- Измерение температуры с разрешением 12 бит (0.0625°C)
- Определение стабилизации: 3 одинаковых показания подряд с допуском ±0.1°C (настраивается в прошивке)
- Звуковой сигнал при стабилизации температуры
- Проверка целостности данных (CRC) при каждом измерении
- Отображение ошибок датчика на экране
- Контроль заряда аккумулятора 18650 с калибровкой
- Обратный отсчёт до следующего измерения
- Диапазон измерения: -55°C до +125°C для оригинального DS18B20 (у меня клон, я бы использовал только в пределах -20...+85°C).

## Поведение

1. При включении — заставка с версией и зарядом батареи на 3 секунды

<img width="460" height="166" alt="Start" src="https://github.com/user-attachments/assets/2b8b0cb8-363f-42ae-bdff-ecd40f839f7e" />

   
3. Основной цикл: измерение каждые 10 секунд
4. На экране: номер измерения и обратный отсчёт (верхняя строка), текущая температура (нижняя)

<img width="458" height="164" alt="image" src="https://github.com/user-attachments/assets/2b0a97bf-3556-47f3-be2d-5b6884f58a3e" />

   
6. После трёх одинаковых измерений (разница ≤0.1°C) — **стабилизация**: экран показывает `OK`, звучит одиночный сигнал

<img width="458" height="163" alt="image" src="https://github.com/user-attachments/assets/eea3459d-2ba9-482a-a70b-45c6f762aa31" />

   
8. При изменении температуры >0.1°C стабилизация сбрасывается
9. Счётчик измерений до 999, затем обнуляется

## Ошибки датчика

| Код | Причина |
|-----|---------|
| `ERR_WAIT` | Датчик не отвечает — проверьте подключение |
| `ERR_CRC` | Ошибка контрольной суммы — помехи на линии |
| `ERR_VAL` | Некорректное значение (-127°C или +85°C) |

При ошибке — тройной звуковой сигнал, пропуск измерения, сброс стабилизации.

## Настройки

Все параметры в блоке `#define` в начале скетча:

```cpp
#define ONE_WIRE_PIN 2       // Пин датчика
#define BUZZER_PIN 3         // Пин бузера
#define BATTERY_PIN A0       // Пин делителя
#define STABLE_OK 3          // Измерений для стабилизации
#define INTERVAL_SEC 10      // Интервал между измерениями
#define MAX_MEAS 999         // Лимит счётчика
#define DRIFT 0.1            // Допуск стабильности (°C)
#define BIP_COUNT 1          // Сигналов при стабилизации

## Примечание
- Несмотря на огромное количество подобных устройств, решил попробовать сделать от себя, делюсь.
- Рекомендую использовать оригинальный DS18B20 (кстати можно проверить скетчем Криса Петрича датчики https://github.com/cpetrich/counterfeit_DS18B20)
- Клоны тоже подойдут, если они сделаны качественно, но я так понял разницу можно не увидеть, особенно в домаших проектах, где точность не важна.
